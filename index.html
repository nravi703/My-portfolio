About the Project

This project marked my very first steps into coding. Using Python's Turtle graphics library, I created a series of creative visualisations — from a Pikachu drawing built entirely out of lines and shapes, to colourful rainbow patterns and geometric flower designs. Each piece explores how simple programmatic commands can be combined to produce surprisingly intricate visual art.

What I Learned

  • The fundamentals of Python syntax, loops, and functions
  • How to use the Turtle library to draw shapes and control movement
  • Translating creative ideas into step-by-step programmatic instructions
  • Working with colour, angles, and coordinates to build visual patterns
