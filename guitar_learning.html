Project Overview

This project is a guitar‑learning rhythm game designed to help beginners practise musical notes, timing, and chord positions. Players follow on‑screen notes, press the correct keys to fret chords, and receive instant feedback based on accuracy.

The game includes:

  • A note‑reading gameplay screen with Perfect/Good/Miss scoring
  • A digital guitar fretboard showing finger positions
  • A GAME OVER screen with leaderboard and score summary
  • Keyboard controls for fretting and finger movement

The aim was to combine interactive gameplay with music education, creating a fun and accessible tool for learning basic guitar skills.
