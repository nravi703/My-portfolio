Magical Library

A 3D Unity game — explore an enchanted library room and collect all the potions, crystals, and scrolls hidden inside.

Unity C# 3D Design Game Dev

About the Project

Magical Library is a 3D first-person game built in Unity using C#. The player is placed inside an enchanted library room filled with mystical objects — glowing crystals, towering potions, and ancient scrolls. The goal is to explore the room and collect all the items before the game ends.

This was built as part of a 3D game design module, focusing on object interaction, player movement, and HUD display in Unity. The environment was designed and scripted entirely from scratch using Unity's game engine and C# scripting.

How to Play

  • Explore the 3D magical library room in first person
  • Walk around and collect all the potions scattered across the room
  • Find and collect all 4 glowing crystals
  • Locate and pick up both ancient scrolls
  • Track your progress using the HUD counter on screen
  • Collect everything to complete the game

Development Challenges

3D environment design in Unity

Designing the library room from scratch in Unity required learning how to place, scale, and light 3D objects effectively to create an atmospheric environment that felt like a real space.

Object collection with C#

Writing the C# scripts to detect when the player walks into a collectible, remove it from the scene, and update the HUD counter in real time was a key technical challenge of the project.

HUD and game state

Displaying the live Potions, Crystals, and Scrolls count on screen required linking Unity's UI system to the game state scripts — making sure the numbers updated correctly each time an item was collected.

What I Learned

This project taught me the fundamentals of 3D game development in Unity — from designing and populating a scene to writing C# scripts for player interaction and UI updates. It gave me hands-on experience with Unity's component system, collider triggers, and canvas UI, which are core skills for any future game development work.
